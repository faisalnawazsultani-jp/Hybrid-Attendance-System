 Hybrid Attendance Model

A modern and responsive front-end interface for a Hybrid Attendance Model. This project demonstrates a clean, user-friendly design for managing attendance in both online and on-campus learning environments.

🚀 Features

* Responsive design for all screen sizes
* Clean and modern user interface
* Student attendance dashboard
* Attendance statistics section
* Interactive navigation
* Optimized layout and performance

🛠️ Built With

* HTML5
* CSS3
* JavaScript

📱 Responsive Design

The website is fully responsive and works smoothly on desktop, tablet, and mobile devices.

 📂 Project Purpose

This project was created to improve front-end development skills by building a realistic education management interface with modern UI/UX principles.
